Typical Request Flows
📝 Create a Note (with image)
1User fills NoteModal, picks an image file
2api.post('/notes', FormData) → fetch with Bearer JWT header
3Express: cors → json → authMiddleware verifies JWT
4multer (memoryStorage) parses file → req.file.buffer
5noteController.createNote() calls uploadToCloudinary(buffer)
6Cloudinary returns secure_url + public_id
7NoteModel.create() inserts row into MySQL notes table
8NoteModel.setTags() inserts into note_tags junction
9Response: 201 { note } → React updates state → card appears
🔐 Login Flow
1User enters email + password on LoginPage
2api.post('/auth/login', { email, password })
3authController.login() → UserModel.findByEmail()
4bcrypt.compare(plain, hashed) validates password
5jwt.sign({ id, email, username }, JWT_SECRET, { expiresIn: 7d })
6Response: 200 { token, user }
7AuthContext stores token in localStorage
8All future requests auto-inject: Authorization: Bearer <token>
9Navigate('/') → PrivateRoute grants access → Dashboard loads
